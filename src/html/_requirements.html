System Requirements

Minimum
  • OS: Windows 10 64 Bit
  • Processor: Intel Core i5-4430 / AMD FX-6300
  • RAM: 8 GB
  • Graphics Card: NVIDIA GeForce GTX 960 TI or AMD Radeon R7 370, with a minimum of 3GB of VRAM
  • DirectX: Version 11
  • Network: Broadband internet connection
  • Disk space: 50 GB
Recommended
  • OS: Windows 10 64 Bit
  • Processor: Intel Core i5-6600K / AMD Ryzen 5 1600
  • RAM: 16 GB
  • Graphics Card: NVIDIA GeForce GTX 1060 / AMD Radeon RX 580 4GB
  • DirectX: Version 11
  • Network: Broadband internet connection
  • Disk space: 50 GB
